Useful API Calls

  • $('h3').slideUp will take all of the <h3> tags in the page and hide them using a slide up effect.
  • $('.awesome').css('color', 'red') will look for anything with the class of awesome and chance the color of the text to red.
  • $('#important').on('click', function () { alert('You clicked me!') } ) will add a click event that alerts the user when they click on the element with the ID of #important.
